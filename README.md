# Examples of Typescript modules
[Les modules en Typescript en 5 min](https://cdiese.fr/typescript-modules/).

Some use cases:
 - [typescript_triple-slash](https://github.com/msoft/typescript_modules/tree/master/typescript_triple-slash): referencing module with triple-slash.
 - [typescript_namespace](https://github.com/msoft/typescript_modules/tree/master/typescript_namespace): implementing modules with namespaces.
 - [typescript_merging_namespace](https://github.com/msoft/typescript_modules/tree/master/typescript_merging_namespace): generating modules with merged namespaces.
 - [typescript_commonjs](https://github.com/msoft/typescript_modules/tree/master/typescript_commonjs): ES5 modules implemented with CommonJS + SystemJS.
 - [typescript_es6](https://github.com/msoft/typescript_modules/tree/master/typescript_es6): ES6 modules.
 - [typescript_es6_amd](https://github.com/msoft/typescript_modules/blob/master/typescript_es6_amd/index.html): ES6 modules generated in AMD format. RequireJS is used for loading modules at runtime.
 - [webpack_es6](https://github.com/msoft/typescript_modules/tree/master/webpack_es6): bundling ES6 modules with webpack.
 - [webpack_es6_namespace](https://github.com/msoft/typescript_modules/tree/master/webpack_es6_namespace): bundling ES6 modules and namespaces with webpack.

For every examples, run:
`npm install && npm run build && npm start`
